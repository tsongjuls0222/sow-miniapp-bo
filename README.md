Tech Stack
_______________
Joi (validation)
JWT Authentication
React (Vite)
Axios

Installation
___________________
git clone <your-repo-url>
cd your-project
npm install

Environment Setup
__________________________
Create .env.development in frontend:

VITE_API_URL=http://localhost:3000

Run Project
_________________________
Backend:
npm run dev

Frontend (separate terminal):
npm run dev

Production

npm run start
